# VoiceFlow Pro 🎙

محول نص لصوت احترافي مجاني بأصوات Microsoft Neural.

## طريقة الرفع على Render (مجاني)

### الخطوة 1: رفع الكود على GitHub
1. افتح github.com وسجل دخول
2. اضغط New Repository
3. اسمه: voiceflow-pro
4. اضغط Create Repository
5. ارفع ملفات المشروع

### الخطوة 2: ربط Render
1. افتح render.com
2. سجل دخول بحساب GitHub
3. اضغط New > Web Service
4. اختار الـ Repository
5. في Start Command اكتب: gunicorn app:app
6. اضغط Deploy

### بعد الرفع
التطبيق هيشتغل على رابط مجاني من Render.

## المميزات
- أصوات عربية وإنجليزية طبيعية
- تنزيل ملف MP3
- تحكم في السرعة والطبقة
- مجاني 100%
